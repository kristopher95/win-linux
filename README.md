# Win2Linux

A simple reference website for people moving from Windows to Linux.

It includes:

- 200 Windows-to-Linux command guides
- A comparison of 50 Linux distributions
- Search and filtering tools
- Detailed distro profiles and official download links
- Light and dark themes

├── index.html
├── linux-distros.html
